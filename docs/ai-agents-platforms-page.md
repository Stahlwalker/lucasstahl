# AI Agents & Platforms Page — Future Page Idea

## Concept

A companion page to the AI Models Guide focused on **AI agent platforms and tools** — the layer that sits on top of models and actually executes tasks autonomously.

The AI Models Guide covers what powers the intelligence. This page would cover what puts that intelligence to work.

## Why a Separate Page

- Agent platforms use models but aren't models themselves
- The space is moving fast — OpenClaw hit 250k GitHub stars in weeks (March 2026)
- Users searching for "best AI agents" have different intent than "best AI models"
- Keeps the models guide focused and authoritative on its topic

## Potential Categories

### 1. Open-Source Autonomous Agents
- **OpenClaw** — Self-hosted autonomous agent with full system access (browser, messaging, scheduling). Originally Clawdbot/Moltbot by Peter Steinberger. Fastest-growing open-source project in history (250k+ GitHub stars as of March 2026). Now under OpenAI / open-source foundation stewardship.
- Others to research: NemoClaw (Nvidia's enterprise variant), community forks

### 2. Cloud AI Agent Platforms
- **Perplexity Computer** — Cloud-based agentic AI that can browse, code, and execute tasks remotely
- **OpenAI Operator** — OpenAI's browser-based agent
- **Base44 Superagents** — No-code agent builder. Describe tasks in natural language, Base44 auto-builds workflows and deploys the agent. One-click integrations (WhatsApp, Telegram, Slack), 24/7 autonomous execution with scheduled jobs and event triggers, persistent memory across sessions. Managed infrastructure with sandboxed environments. Launched March 2026.
- Others to research as space evolves

### 3. Local/Hardware AI Agents
- **Perplexity Personal Computer** — Always-on local agent running on a dedicated Mac mini with full file/app access. Positioned as more secure than OpenClaw (user confirmation required, built-in audit trail). Launched March 2026 via waitlist.
- Others to research as category develops

### 4. Developer-Focused Agents
- **Claude Code** (Anthropic) — CLI-based coding agent
- **Cursor Agents** — Background agents triggered from GitHub PRs, Slack, Linear
- **Replit Agent 4** (Replit) — Full-stack app builder with parallel agents working on different app components simultaneously. Infinite canvas for visual design exploration, Trello-style task board for agent progress, one-click deploy. Targets both non-technical users and developers. Launched March 2026.
- **Gemini CLI** (Google) — Terminal-based coding agent
- **Kimi Code CLI** (Moonshot AI) — Open-source coding agent
- **GitHub Copilot Workspace** — PR-to-code agent

### 5. Enterprise Agent Frameworks
- To be researched — LangChain, CrewAI, AutoGen, etc.

## Key Differentiators to Track Per Entry

- Open-source vs. closed
- Local vs. cloud execution
- What models it supports/uses under the hood
- System access level (sandboxed vs. full)
- Security model (auto-execute vs. user confirmation)
- Pricing (free, subscription, API costs)
